# Google Sheets + Apps Script

## 1. Создать таблицу

1. [Google Таблицы](https://sheets.google.com) → новая таблица.
2. Скрипт привязывать **из этой таблицы**: **Расширения** → **Apps Script** (не отдельный проект на script.google.com).

## 2. Установить скрипт

1. Вставьте код из [`Code.gs`](Code.gs).
2. Сохраните (Ctrl+S).

## 3. Развернуть веб-приложение (важно!)

1. **Развернуть** → **Новое развёртывание**.
2. Тип: **Веб-приложение**.
3. **Описание:** любое (например `rsvp-v1`).
4. **Запуск от имени:** **Я** (ваш email).
5. **У кого есть доступ:** **Все** — именно «Все», не «Только я».

   | Правильно | Неправильно |
   |-----------|-------------|
   | Все | Только я |
   | Все | Только пользователи организации |

6. **Развернуть** → пройти авторизацию (окно Google «приложение не проверено» — это нормально, см. «Дополнительные настройки»).
7. Скопировать URL: `https://script.google.com/macros/s/...../exec`

### Если форма не работает после первого развёртывания

1. **Развернуть** → **Управление развёртываниями**.
2. Нажать карандаш (изменить) у активного развёртывания.
3. Снова выбрать **У кого есть доступ: Все**.
4. **Версия:** «Новая версия» → **Развернуть**.
5. URL `/exec` обычно не меняется — вставьте его в `script.js`.

## 4. Подключить к сайту

В `script.js`:

```javascript
GAS_WEB_APP_URL: 'https://script.google.com/macros/s/ВАШ_ID/exec',
```

## 5. Проверка

Откройте URL в браузере (без входа в Google). Должно показать:

```json
{"ok":true,"status":"ready"}
```

Если видите **«Нет доступа»** — доступ всё ещё не «Все». Переразверните.

Тест отправки (PowerShell):

```powershell
curl.exe -X POST "ВАШ_URL" -H "Content-Type: text/plain;charset=utf-8" -d '{\"firstName\":\"Тест\",\"lastName\":\"Тестов\",\"attendance\":\"yes\",\"genderVote\":\"boy\"}'
```

Ответ: `{"ok":true}` — в таблице появится лист `responses` с заголовками.

## Русские заголовки (вручную в таблице)

Скрипт **не перезаписывает** строку 1, если лист уже с данными. Замените заголовки сами в ячейках A1–F1:

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| Дата и время | Имя | Фамилия | Придёт? | Прогноз | Служебный ключ |

Столбец F можно скрыть: правый клик по F → **Скрыть столбец**.

На работу формы это **не влияет** — скрипт смотрит на номера столбцов, не на текст заголовков.

Диаграмма на листе **«Статистика»** обновляется при каждой новой отправке.
